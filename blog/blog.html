var: The var statement declares a variable. it is an old version of the js variable declaration
let: let allows you to declare variables that are limited to the scope of a block statement, or expression on which it is used, unlike the var keyword, which declares a variable globally, or locally to an entire function regardless of block scope. It is an ES6 feature.
Const: Constants are block-scoped, much like variables declared using the let keyword. The value of a constant can't be changed through reassignment, and it can't be redeclared. It is an ES6 feature.

Arrow functions are a new feature introduced in ES6 enables writing concise functions in JavaScript. While both regular and arrow functions work in a similar manner, there are specific interesting differences between them, as discussed below.
Regular functions created using function declarations or expressions are constructible and callable. Since regular functions are constructible, they can be called using the new keyword.
However, the arrow functions are only callable and not constructible. Thus, we will get a run-time error when constructing a non-constructible arrow function using the new keyword.

map() calls a function once for each element in an array.
forEach() method calls a function for each element in an array.
filter() method creates a new array filled with elements that pass a test provided by a function.
find() method returns the value of the first element that passes a test.
